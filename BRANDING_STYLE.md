# StoryCreator Brand Style Guide

## Design System
The visual identity is clean, minimal, and confident. The brand uses a warm off-white background with bold black typography and a signature gold/amber accent color.

## Color Palette

| Role | Hex Code | Usage |
|---|---|---|
| Primary Gold / Amber | `#F5A623` | CTAs, active states, highlights, brand accent |
| Dark / Black | `#1A1A1A` | Headlines, body text, logo `.Bot` |
| Medium Gray | `#555555` | Body copy, descriptions |
| Light Gray Background | `#F5F5F5` / `#FAFAF8` | Page backgrounds, card fills |
| Border Gray | `#DDDDDD` | Card borders, dividers, input borders |
| White | `#FFFFFF` | Card backgrounds, input fields |
| Partner Banner BG | `#1A1A1A` (dark) | Verified Business Partner section background |

## Typography
- Font Family: System sans-serif stack (Inter or similar). Logo uses mixed weights.
- Logo: `StoryCreator` in gold/amber bold, `.Bot` in dark/black bold.
- Hero Headline: Very large, bold, dark. Key phrase in gold.
- Body Text: Medium gray, comfortable line height.
- Buttons: Bold, rounded corners (`border-radius: ~8px`).

## Button Styles
- Primary CTA: Gold/amber gradient background (`linear-gradient(to right, #FFC837, #F5A000)`), dark text, bold, rounded. Example: `Get Started`, `Next`.
- Secondary / Outline: White background, dark border and text, rounded. Example: `Test Drive the Bot`, `Log In`.
- Destructive / Neutral: Light gray background for less prominent actions.
