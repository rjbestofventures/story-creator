Dictation Guide

Use Dictation To Answer

⌨️ Click in the answer box, then use the shortcut for your device
Mac users Click inside the answer box, then press fn twice to start speech to text.
Windows users Click inside the answer box, then press Windows + H to start speech to text.

Mac Guide

1

Click inside the answer box

Place your cursor in the StoryCreator.Bot answer field.

2

Press fn twice

Click inside the answer box, then press fn twice to start speech to text.

Note: Dictation must be turned on in System Settings → Keyboard → Dictation.
3

Start speaking

Your Mac will turn your voice into text directly in the answer box.

4

Review your answer

Check the text, make any edits, then send your answer in StoryCreator.Bot.

That’s it. Mac users can start speech to text by clicking inside the answer box, then pressing fn twice.

Windows Guide

1

Click inside the answer box

Place your cursor in the StoryCreator.Bot answer field.

2

Press Windows + H

Click inside the answer box, then press Windows + H to start speech to text.

3

Start speaking

Windows will turn your voice into text directly in the answer box.

4

Review your answer

Check the text, make any edits, then send your answer in StoryCreator.Bot.

That’s it. Windows users can start speech to text by clicking inside the answer box, then pressing Windows + H.
